# Citations

RateXp is built on the open-source projects and services listed below. Where a
project publishes its own preferred citation (a `CITATION.cff` or equivalent), that
form is used; otherwise a standard software-citation format is followed:

> Author / Organization. *Title* (version, if used) [Software]. URL.

## Projects with a requested citation

**FastAPI** - follows the project's [`CITATION.cff`](https://github.com/fastapi/fastapi/blob/master/CITATION.cff):

> Ramírez, S. *FastAPI* [Software]. https://github.com/fastapi/fastapi - https://fastapi.tiangolo.com

**LangChain** - follows the project's [`CITATION.cff`](https://github.com/langchain-ai/langchain/blob/master/CITATION.cff) ("If you use this software, please cite it as below"):

> Chase, H. (2022). *LangChain* [Software]. https://github.com/langchain-ai/langchain

## ATIF / Harbor

RateXp stores transcripts in **ATIF - the Agent Trajectory Interchange Format
(Harbor)**, a JSON shape for a whole agent conversation. If you have the canonical
upstream reference for ATIF/Harbor, cite it in the same software-citation format
above.
